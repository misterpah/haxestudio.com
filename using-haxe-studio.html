To run Haxe Studio on windows, just double click the batch file windows.bat

 

To run Haxe Studio in linux, just double click the SH file linux.run

Using the Haxe Studio

The first thing you can (should) do is open a Haxe/OpenFL Project. Haxe Studio recognize the project file by the extensions (.hxml & .xml) .

As for example, We will be using HaxeFlixel project. HaxeFlixel is an awesome game library ported from the famous flash library, Flixel. You can read more about how to create HaxeFlixel projects here.

Open a project

press CTRL + SHIFT + O to Open Project . Alternatively, you may click from the menu ( File -> Open Project )

If the project file is valid Haxe/OpenFL project, Haxe Studio will open the Project Tree (this is a feature added by a plugin named plugin.misterpah.ProjectTree )

 

You can close the ProjectTree by clicking HIDE.

As For now [January 17,2014] the explorer inside ProjectTree plugin aren’t complete yet.

Editing File with Autocomplete features

Autcomplete in Haxe are provided by the Haxe Server. To make sure you can use the autcomplete, make sure you’ve opened the related project file. 

To use Autocomplete, you must save the file changes first. That is the Official way to obtain autocomplete from Haxe Toolkit.

First, you need to open a file for editing. Simply press CTRL + O to Open a File . Alternatively, you may click from the menu ( File -> Open File)

 

Currently, Autocomplete are invoked automatically when you press dot ( . )

 

As you typed , autocomplete will change accordingly.
