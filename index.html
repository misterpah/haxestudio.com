Haxe Studio

Open Source Integrated Development Environment (IDE) for Haxe.

Multiplatform

Haxe Studio is developed using Haxe (Javascript target) , Javascript library (mostly jQuery) and the famous application runtime node-webkit. These technologies enable Haxe Studio to executed on Linux (32 & 64 bit) , Windows and OSX.

Open Source

Haxe Studio will always be an Open Source and free to use. Visit our Github page to get the source.
