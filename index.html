Haxe Studio

Open Source Integrated Development Environment (IDE) for Haxe.


Now, Haxe studio supports:

Multiplatform

Haxe Studio is developed using Haxe (Javascript target) , Javascript library (mostly jQuery) and the famous application runtime node-webkit. These technologies enable Haxe Studio to executed on Linux (32 & 64 bit) and Windows.


Although untested, we believe Haxe Studio also can be executed on OSX.
** update (2 March 2014) **
we've test Haxe Studio (master-branch) in OSX!

Open Source

Haxe Studio will always be an Open Source and free to use. Visit our Github page to get the source.
